# Known Element Android Classic Bugs

Bugs in Element Android Classic that Progressive Chat's C++ port intentionally
fixes rather than inherits.

## Reaction Count Miscounting

Element Classic sometimes shows wrong reaction counts under messages.
The count can be off by 1-3 or flicker during sync.

## Context Menu Not Loading

The long-press context menu on messages occasionally fails to load completely
— buttons for "Edit", "Reply", "Copy" etc. are missing or unresponsive.
